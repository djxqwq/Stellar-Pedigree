# 🎮 测试指南 - 完整玩家控制器

## 当前功能测试

### ✅ 已实现功能
- **WASD移动** - 八方向平滑移动
- **鼠标射击** - 按住左键连续射击
- **生命值系统** - 受伤、治疗、无敌时间
- **视觉效果** - 飞船形状、子弹粒子效果、星空背景
- **基础武器** - 脉冲步枪（支持3形态进化）

### 🎯 测试方法

#### 1. 打开游戏
1. 用 Godot 4.2+ 打开项目
2. 运行 `src/scenes/Main.tscn`
3. 或按 F5 运行主场景

#### 2. 移动测试
- **WASD** - 八方向移动
- 观察移动是否流畅
- 松开按键是否有平滑停止效果

#### 3. 射击测试
- **鼠标左键** - 按住连续射击
- 子弹朝鼠标方向发射
- 观察子弹粒子效果

#### 4. 生命值系统测试
- **Enter键** - 模拟受到10点伤害
- **Shift键** - 模拟治疗20点生命
- 观察受伤后的无敌闪烁效果

#### 5. 游戏控制
- **ESC键** - 暂停/继续游戏

### 📊 控制台输出
游戏会在控制台显示：
- "Stellar Pedigree - Game Started"
- 生命值变化
- 伤害和治疗信息
- 死亡信息

### 🎨 视觉效果
- 青色三角形飞船
- 黄色子弹带粒子效果
- 闪烁星空背景
- 受伤无敌闪烁
